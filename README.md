## Controls/Instructions

### Menu Controls:
- SPACE BAR = PAUSE/START 
- X = ON THE TOP OF THE WINDOW TO EXIT THE SCREEN 
### Movements:
- W = UP
- S = DOWN
- A = LEFT
- D = Right
- Mouse Wheel = ZOOM IN / ZOOM OUT
